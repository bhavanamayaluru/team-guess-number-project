# Guess the Number Game
> A simple number guessing game for CSCE1015

## About
This project is a browser-based number guessing game where the player tries to guess a randomly generated number between 1 and 100. It was built as part of CSCE1015 to practice Git workflows, Agile methodology, and project documentations

## Features
- Player can start a new game!
- Player can enter guesses!
- Game gives hints if the guess is too high or too low!
- Tracks number of attempts!
- Displays a winning message when the correct number is guessed!

How to Play

1. Click “Start Game”

2. Enter your guess in the input box

3. Click “Submit”

4. Read the hint: "Too high", "Too low", or "Correct!"

5. Repeat until you guess the correct number

Running Tests

- Enter numbers 1–100 and check hints

- Enter invalid inputs (letters, empty input) and check for alerts

- Check that number of attempts increments correctly

- Confirm game resets properly after winning
